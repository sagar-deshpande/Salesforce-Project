# Salesforce-Project

This project helps an event management company to manage their events very effectively along with the attends and location information. 

We need following custom objects:

1. Location:
   1. Street (Text)
   2. City (Text)
   3. State (Text)
   4. Postal code (Text) 
   5. Country (Picklist)
   6. Landmark (Text)
   7. Verified (Checkbox)

2. Event Organizer
3. Event
4. Attendees
5. Speaker
6. Event-Attendee
7. Event-Speaker
8. Error-log
